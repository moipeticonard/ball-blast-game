# Ball Blast Game 🎮

Un jeu mobile ultra-simple : contrôle une balle et tire sur des ennemis au style Block Blast.

## Caractéristiques

✅ Fond noir complet - Plein écran immersif
✅ Gameplay arcade - Contrôle tactile fluide
✅ Pas de menu - Direct au jeu !
✅ Ennemis progressifs - Score à augmenter

## Installation

```bash
git clone https://github.com/moipeticonard/ball-blast-game.git
cd ball-blast-game
./gradlew assembleRelease
```

L'APK sera générée dans `app/build/outputs/apk/release/`

## Contrôles

- **Doigt sur l'écran** : Déplace ta balle
- **Tap** : Tire un projectile
